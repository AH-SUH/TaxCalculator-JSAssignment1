# 💰 Tax Calculator

A simple JavaScript tax calculator that computes income tax based on U.S. federal tax brackets.

## 📌 Description

This project allows users to enter a taxable income amount and calculates the income tax owed using the first two tax brackets.

It demonstrates basic JavaScript concepts including:

- DOM manipulation
- Event handling
- Conditional logic
- Number conversion and formatting

## 🛠️ Technologies Used

- HTML
- JavaScript

## ⚙️ How It Works

1. Enter a taxable income amount
2. Click the **Calculate** button
3. The calculated tax is displayed on the page

## 📊 Tax Brackets Used

- **$0 – $9,275** → 10% of income
- **$9,275 – $37,650** → $927.50 + 15% of amount over $9,275

## 🧪 Example Inputs

| Income | Tax Owed |
| ------ | -------- |
| 9275   | 927.50   |
| 37650  | 5183.75  |

## 🚀 Features

- Validates user input
- Rounds tax to two decimal places
- Simple and responsive interface
